## Part 2 Improvements

### Responsive Design

* Added tablet and mobile responsive layouts using media queries.
* Improved navigation responsiveness for smaller screens.
* Added flexible layouts using Flexbox.
* Converted fixed pixel values into relative units such as rem and percentages.
* Added responsive image attributes using srcset and sizes.

### CSS Enhancements

* Improved typography and spacing across all pages.
* Added hover effects and transitions to buttons and navigation links.
* Styled the enquiry form professionally.
* Improved product card responsiveness and visual appearance.
* Added consistent layout spacing and responsive containers.

### Testing
